# PyeongChang Olympics Android Application

-  This is Android Application for introduction about each Olympic Games and Korean players for celebrating 2018 PyeongChangOlympics.

## Development Environment
- IDE : Android Studio
- OS : Windows10
- Language : Java
- DB : Firebase
